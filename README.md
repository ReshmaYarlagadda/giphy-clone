# Giphy Clone 🎬

A simple and responsive Giphy Clone built using HTML, CSS, and JavaScript. This project allows users to search and explore GIFs using the Giphy API.

## Features

* Search GIFs instantly
* View trending GIFs
* Clean and responsive UI
* Real-time API data fetching

## Tech Stack

* HTML
* CSS
* JavaScript
* Giphy API

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
